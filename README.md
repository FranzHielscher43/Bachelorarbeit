# 🎮 Serious Game: Objektorientiertes Paradigma

## 📖 Überblick

Dieses Projekt entsteht im Rahmen meiner Bachelorarbeit im Studiengang Medieninformatik.

Ziel ist die Entwicklung einer **interaktiven Anwendung zur Vermittlung des objektorientierten Paradigmas (OOP)**. Der Schwerpunkt liegt auf der **didaktischen Aufbereitung und interaktiven Vermittlung zentraler OOP-Konzepte**.

Link zu Overleaf: https://de.overleaf.com/project/69df70deb01344326624fe1a

Link zu Figma: https://www.figma.com/files/team/

---

## 🎯 Zielsetzung

Die Anwendung soll Studierenden:

- einen praxisnahen Einstieg in OOP ermöglichen
- abstrakte Konzepte verständlich darstellen
- Lernen durch Interaktion fördern

---

## 🧠 Lerninhalte

Im Fokus stehen grundlegende Konzepte der objektorientierten Programmierung:

- Klassen und Objekte
- Attribute und Methoden
- Kapselung
- Vererbung
- Polymorphie
